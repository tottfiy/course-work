# Automated Blue Team Hardening Platform for CTF Environments
Клієнт-серверний інструмент для автоматизованого інтерактивного менеджменту безпеки локальної інфраструктури, який забезпечує:
* швидке розгортання
* просте управління загрозами
* можливість власної конфігурації
* аналіз загроз trusted інструментами (nmap, linenum, sudoscan etc.)
* дешборд огляд статусу інфраструктури
